# Фабричный метод

>Реализация паттерна - "Абстрактная Фабрика" - учебный репозиторий.

> Идея - Фабрика собирает телефоны.

### Файлы

- client_code - запуск программы
- interfaces - интерфейс
- products - продукты для телефонов
- phones - сборка телефонов
